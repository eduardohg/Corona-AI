Corona AI

Corona AI é um agente inteligente criado usando processamento de linguagem natural e inteligência artificial para tratar do assunto Coronavirus.

É um projeto da Universidade Estadual de Londrina (UEL) que visa o desenvolvimento de tecnologia para difusão e avaliação de veracidade em notícias e sanar dúvidas para o combate à epidemia do Coronavirus.

Espera-se desta forma mitigar o efeito nocivo de mensagens e notícias falsas (fake news), bem como responder dúvidas e questionamentos da sociedade a cerca do Coronavirus.

Para começar, mande suas perguntas para o CoronaAI ao lado.

Descrição

Base de Dados

Contamos com uma base de dados que será o arcabouço no qual diferentes documentos serão armazenados e disponibilizados para extração de informação estruturada para consulta e treinamento do CoronaAI.

Troca de Mensagens

O CoronaAI funciona como um ChatBot para o esclarecimento de dúvidas e questionamentos da população sobre o Coronavirus. Com apenas uma pergunta ele pode te ajudar.

Fake News

Este sistema utilizará uma plataforma construída sob bases confiáveis. Considerando que os sistemas de mensagens instantâneas estão entre os meios de maior propagação de notícias falsas, a proposta deste projeto está vinculada a utilizar tal solução tecnológica como interface para validação das mais diversas notícias.
